S1
-----------------------------------------------------------
enable
configure terminal
hostname S1
ip domain-name ccna.coma
vlan 10 
name Red
vlan 20
name Green
vlan 30
name  Blue
vlan 999
name Native

interface range g0/1-2
switchport mode trunk
switchport trunk native vlan 999
interface g0/1
switchport mode dynamic desirable
interface g0/2
switchport mode trunk
switchport nonegotiate
exit

exit
copy running-config startup-config
show dtp
show interface trunk
show vlan brief
